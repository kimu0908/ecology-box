## 下一阶段计划

### Phase 4.1：轻量动画

目标：让三只动物更有生命感，但不破坏状态机。

建议方向：

- 河豚：轻微呼吸 / 压扁后的果冻回弹。
- 章鱼：轻微漂浮 / 左右摆动 / 黏液状态过渡更自然。
- 刺猬：轻微呼吸 / 警戒时轻微抖动 / 慢慢靠近成功时小幅放松。
- UI：提示框淡入淡出，按钮微动效。

限制：

- 不做复杂骨骼动画。
- 不改触发条件。
- 不改距离阈值。
- 不改核心状态机。

### Phase 5：AI-like 导师翻译机制

目标：把动物反应翻译成儿童能理解的现实社交规则。

建议先做规则驱动的导师卡片，不直接接真正 AI API。

可新增函数：

```js
showMentorTip(type)
```

示例：

- `pufferfish_too_close`
- `octopus_need_space`
- `hedgehog_slow_success`
- `hedgehog_too_fast`

导师卡片只在关键成功 / 失败节点出现，不要频繁打扰。

### Phase 6：图鉴式总结奖励

目标：当儿童完成正确互动后，获得一张小卡片，总结学到的边界规则。

示例：

- 河豚卡：靠太近会挤到别人。
- 章鱼卡：先退后，再修复互动。
- 刺猬卡：慢慢靠近，让朋友安心。
