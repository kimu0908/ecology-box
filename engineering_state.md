## 已完成内容

### 1. 启动页

- 已有完整启动页。
- 支持手机端和 iPad 端显示。
- 有开始体验按钮。
- 有角色视觉入口。
- 有声音开关。
- iPad / 手机端均已测试可进入 AR。

### 2. AR 放置流程

- WebAR 地面扫描可用。
- reticle / 绿色放置圈可显示。
- 点击地面后可放置模型。
- 模型放置后固定在世界坐标，不再跟随 camera。
- 支持重新放置。
- 支持返回首页后再次进入体验。
- 手机端和 iPad 端均已完成测试。

### 3. 三个动物模型

- `pufferfish.glb` / `octopus.glb` / `hedgehog.glb` 均可正常显示。
- 三个模型可在同一个 anchor 位置切换。
- GLB src 未改变：
  - `#asset-pufferfish`
  - `#asset-octopus`
  - `#asset-hedgehog`

### 4. 河豚 Pufferfish 交互

- 距离过近时触发压扁。
- 退后后恢复。
- 交互意义：柔性与弹性边界。
- 教育隐喻：可以靠近一点，但不要过度挤压对方空间。
- 手机端 / iPad 端核心交互均已测试完成。
- 手机端动物触发音效可用。
- iPad 端动物触发音效存在设备差异，暂不继续修。

### 5. 章鱼 Octopus 交互

- 距离 `< 0.5m` 时触发黏液 / stuck。
- 需要退到 `0.8m` 外后才可甩动解除。
- DeviceMotion 甩动逻辑可用。
- 黏液 overlay 可显示。
- 解除后恢复正常。
- 交互意义：高渗透 / 融合边界。
- 教育隐喻：过度热情也需要重新找回彼此空间。
- 手机端 / iPad 端核心交互均已测试完成。

### 6. 刺猬 Hedgehog 交互

- 快速靠近可触发警戒。
- 慢慢靠近可触发安心提示。
- 警戒后需要退到安全距离并等待，才可恢复。
- 已修正“放置太近直接成功”的问题。
- 已修正“还没退后就提示距离足够”的文案逻辑。
- 交互意义：刚性与隐形边界。
- 教育隐喻：谨慎的朋友需要慢慢靠近与等待。
- 手机端 / iPad 端核心交互均已测试完成。
- 红色警戒视觉最终以 UI 文案警戒为主，不继续强依赖 GLB 材质变色。

### 7. instruction-text / 导师提示

- 交互提示框已完成。
- 关键提示具备锁定时间，避免被默认文案快速覆盖。
- 过程提示用于指导儿童下一步动作。
- 成功提示用于解释正确边界行为。
- 当前版本不再继续扩展外部 AI API。

### 8. learning card / 学习卡

- 三个动物完成正确互动后会弹出学习总结卡。
- 学习卡不会自动消失。
- 用户点击“我知道了”后关闭。
- 学习卡用于完成“尝试 → 犯错 → 理解 → 记忆”的教育闭环。

### 9. 边界图鉴 / 收藏系统

- 已新增边界图鉴。
- 三个动物均有图鉴卡：
  - 河豚｜柔性边界
  - 章鱼｜融合边界
  - 刺猬｜隐形边界
- 完成对应互动后解锁“已学习”状态。
- 手机端图鉴布局已适配。
- iPad 端图鉴布局已适配。
- `localStorage` 可保存图鉴学习状态。

### 10. 声音系统

- 已加入启动页 BGM / UI 声音 / 部分交互音效。
- 手机端动物交互音效可用。
- iPad 端动物交互音效存在设备播放差异，可能与 iPadOS Safari / WebXR / Audio unlock / cloneNode 播放策略有关。
- 当前决定：iPad 端动物音效差异暂不继续修复，不影响核心交互与作品展示。
- 后续如需继续提升，可改为预加载单实例 audio、WebAudio 管理或预录导师语音。

### 11. 暂缓外部 AI API

- 当前版本不接 OpenAI API。
- 不接外部 LLM。
- 不接语音识别。
- 不做开放式 AI 对话。
- 目前以规则驱动的导师提示、学习卡、边界图鉴表达 AI-like 导师机制。
- Future Work 可写：后续接入生成式语音导师 / OpenAI TTS / 个性化解释系统，但需要后端代理，不能把 API Key 放在 GitHub Pages 前端。
